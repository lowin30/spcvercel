# sequential thinking (pensamiento secuencial)

esta skill permite al agente abordar problemas complejos dividiendolos en pasos logicos secuenciales, evaluando alternativas, detectando errores y corrigiendo el rumbo de forma transparente.

## cuando aplicar
*   al realizar refactorizaciones complejas que involucren multiples archivos.
*   al resolver discrepancias de seguridad o politicas rls en la base de datos.
*   al planificar cambios arquitectonicos que requieran coherencia extrema.
*   cuando se necesite deliberar antes de ejecutar acciones irreversibles.

## como usar
1.  **dividir el problema:** identificar los componentes clave del problema.
2.  **formular hipotesis:** plantear posibles soluciones o explicaciones.
3.  **verificar suposiciones:** contrastar las hipotesis con el codigo real de la aplicacion.
4.  **revisar y corregir:** si una hipotesis resulta ser falsa o conflictiva, retroceder y ajustar el rumbo de inmediato.
